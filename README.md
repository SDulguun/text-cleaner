# Text Cleaner

A single-page text cleaning tool built with vanilla HTML, CSS, and JavaScript.

## Features

- Text cleaning operations (trim, collapse spaces, remove empty lines, remove duplicates, unwrap paragraphs, fix punctuation, smart quotes)
- Live stats bar (characters, words, sentences, lines, paragraphs)
- Case conversion (sentence, title, upper, lower)
- Copy, clear, and undo actions
- Export/download cleaned text as .txt file
- Import text from file (file picker + drag-and-drop)
- localStorage persistence (auto-saves text and settings across sessions)
- Dark mode toggle with system preference detection
- Before/after diff preview
- Responsive layout for mobile

## Usage

Open `index.html` in any modern browser. No build tools or dependencies required.
