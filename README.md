# Text Cleaner

A single-page text cleaning tool built with vanilla HTML, CSS, and JavaScript.

## Features (Planned)

- Text cleaning operations (trim, collapse spaces, remove empty lines, etc.)
- Live stats bar (character count, word count, etc.)
- Case conversion options
- Copy, clear, and undo actions
- Dark mode toggle
- Before/after diff preview
- Responsive layout for mobile

## Usage

Open `index.html` in any modern browser. No build tools or dependencies required.
